# 🐍 Snake Water Gun Game

A simple command-line implementation of the classic **Snake, Water, Gun** game using Python. The player competes against the computer in a best-of-5 match, where the computer makes random choices and the game determines the winner based on predefined rules.

## 🎮 Game Rules

* 🐍 Snake drinks Water → Snake Wins
* 💧 Water damages Gun → Water Wins
* 🔫 Gun kills Snake → Gun Wins

## ✨ Features

* Random computer choice using Python's `random` module.
* Interactive command-line gameplay.
* Best-of-5 rounds.
* Real-time score tracking.
* Displays the final winner after all rounds.
* Input validation for incorrect choices.

## 🛠️ Technologies Used

